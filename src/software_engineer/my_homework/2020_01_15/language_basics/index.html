▼ Code ▼

//--▼ Multiple Operators ▼--//

var fullAge = 18;

//--▲ More Operators ▲--//

▼ Notes ▼

Operator precedence determines how operators are parsed concerning each other. Operators with higher precedence become the operands of operators with lower precedence.
